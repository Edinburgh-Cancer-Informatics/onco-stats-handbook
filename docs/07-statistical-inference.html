9 Concepts in Statistical Inference

9.1 The Logic of Statistical Inference

Consider this historical example:

Until the end of the 17th century, Europeans assumed all swans were white. This hypothesis was rejected when Willem de Vlamingh spotted a black swan in Australia in 1697.

The null hypothesis (“All swans are white”) was assumed true until evidence proved otherwise.

Statistical inference follows the same logic:

  1. Start with a hypothesis
  2. Assume it is true
  3. Collect data
  4. Seek evidence to reject or fail to reject it

9.2 Hypotheses

9.2.1 The Null Hypothesis (H₀)

The null hypothesis is a statement of “no difference” or “no effect.”

We assume H₀ is true unless the evidence strongly suggests otherwise.

Example: In a clinical trial comparing a new drug to standard treatment:

H₀: There is no difference in blood pressure between the new drug and standard treatment

9.2.2 The Alternative Hypothesis (H₁)

The alternative hypothesis is what we suspect might be true – the opposite of H₀.

H₁: There is a difference in blood pressure between the drugs

Why Start with “No Effect”?

We begin by assuming no effect because it’s easier to disprove something than to prove it. One black swan disproves “all swans are white,” but millions of white swans can never prove it.

9.3 The P-value

After collecting data, we calculate a test statistic and convert it to a P-value.

9.3.1 What is a P-value?

Definition

The P-value is the probability of observing results as extreme as (or more extreme than) what we found, assuming the null hypothesis is true.

9.3.2 Interpreting P-values

P-value Interpretation
Small (e.g., P < 0.05) Results are unlikely if H₀ were true → Evidence against H₀
Large (e.g., P > 0.05) Results could easily occur if H₀ were true → Insufficient evidence against H₀

9.3.3 The 0.05 Threshold

By convention, P < 0.05 is considered statistically significant.

  • P < 0.05: Reject H₀ in favour of H₁ – results are “statistically significant”
  • P ≥ 0.05: Cannot reject H₀ – results are “not statistically significant”
Important!

P > 0.05 does not mean H₀ is true. It means we don’t have enough evidence to reject it.

9.4 One-tailed vs Two-tailed Tests

9.4.1 Two-tailed Test

The alternative hypothesis does not specify a direction:

H₁: The drugs have different effects (could be better OR worse)

This is the standard approach.

9.4.2 One-tailed Test

The alternative hypothesis specifies a direction:

H₁: The new drug is better than the standard drug

One-tailed tests are generally discouraged because we rarely know beforehand which direction the effect will be.

9.5 Type I and Type II Errors

When making decisions, we can make two types of mistakes:

Show R code 
error_table <- tibble(
  ` ` = c("**Reject H₀**", "**Do not reject H₀**"),
  `H₀ is actually TRUE` = c("Type I error\n(false positive)", "✓ Correct decision"),
  `H₀ is actually FALSE` = c("✓ Correct decision", "Type II error\n(false negative)")
)

kable(error_table, align = "c")

9.5.1 Type I Error (False Positive)

  • Concluding there is an effect when there isn’t one
  • Probability = α (alpha) = significance level (usually 0.05)

Example: Approving an ineffective drug

9.5.2 Type II Error (False Negative)

  • Concluding there is no effect when there is one
  • Probability = β (beta)

Example: Dismissing an effective treatment

9.5.3 Power

Power = 1 - β = Probability of correctly detecting a real effect

Power is the chance of rejecting H₀ when it is actually false (i.e., detecting a true effect)

9.6 Confidence Intervals vs P-values

Both provide information about statistical significance, but confidence intervals offer more:

P-value Confidence Interval
Tells us if result is significant Tells us if result is significant
Doesn’t show effect size Shows plausible range of effect sizes
Binary (significant or not) Continuous (narrow = precise, wide = imprecise)

9.6.1 Using Confidence Intervals

If the 95% CI for a difference excludes zero:

  • The difference is statistically significant (P < 0.05)
  • We have evidence of a real effect

Example: New drug reduces blood pressure by 8 mmHg (95% CI: 3 to 13 mmHg)

  • The CI excludes 0 → Statistically significant
  • We can be confident the true effect is between 3 and 13 mmHg

9.7.1.1 Parametric Tests (Normal data)

Situation Test
One sample vs known value One-sample t-test
Two independent groups Two-sample t-test
Two paired measurements Paired t-test

9.7.1.2 Non-parametric Tests (Non-Normal data)

Situation Test
One sample vs known value Wilcoxon signed-rank test
Two independent groups Mann-Whitney U test (Wilcoxon rank-sum)
Two paired measurements Wilcoxon signed-rank test

9.7.2 Tests for Categorical Data

Situation Test
Unpaired, large samples Pearson chi-squared (χ²) test
Unpaired, small samples (expected counts < 5) Fisher’s exact test
Paired categorical data McNemar’s test

9.8 Sample Size and Power

9.8.1 Why Sample Size Matters

A study needs enough participants to have a reasonable chance of detecting a real effect.

9.8.2 Factors Affecting Power

Factor Effect on Power
↑ Sample size ↑ Power
↑ Variability (SD) ↓ Power
↑ Effect size sought ↑ Power
↑ Significance level (α) ↑ Power

9.8.3 Underpowered Studies

An underpowered study (too few participants) may fail to detect a real effect:

  • Important findings may be missed
  • Negative results are inconclusive
  • Can be considered unethical

9.8.4 Overpowered Studies

An overpowered study (more participants than needed):

  • Wastes resources
  • May detect trivially small effects that aren’t clinically meaningful
  • Exposes unnecessary participants to research procedures

9.9 Clinical vs Statistical Significance

A Critical Distinction

Statistical significance (P < 0.05) does not mean clinical significance.

A statistically significant result may be too small to matter clinically.

Example: A new drug lowers blood pressure by 2 mmHg (P = 0.001)

  • Statistically significant? Yes (P < 0.05)
  • Clinically significant? Probably not (2 mmHg is unlikely to reduce cardiovascular events)

Always consider:

  1. Is the result statistically significant?
  2. Is the effect size clinically meaningful?
  3. Is the confidence interval narrow enough to be useful?

9.10 Summary

  • The null hypothesis (H₀) assumes no effect; we seek evidence to reject it
  • A P-value is the probability of the observed results if H₀ were true
  • P < 0.05 is conventionally “statistically significant”
  • Type I error: concluding an effect exists when it doesn’t
  • Type II error: missing a real effect
  • Power: probability of detecting a true effect
  • Confidence intervals provide more information than P-values alone
  • Choose tests based on data type and distribution
  • Always consider clinical significance, not just statistical significance
