7 Descriptive Measures

7.1 Overview

A descriptive measure is a numerical value that summarises a dataset.

We typically need two types of measures:

  1. Measures of location (central tendency) – Where is the centre?
  2. Measures of dispersion (spread) – How spread out are the values?

7.2 Notation

Before we begin, some standard notation:

Symbol Meaning
n Number of observations in the sample
x The variable of interest
x₁, x₂, …, xₙ Individual values for each patient
Σ Sum (add together)

7.3 Measures of Location

These describe the centre or typical value of the data.

7.3.1 Mean (Average)

The sample mean (denoted x̄, pronounced “x-bar”) is the sum of all values divided by the number of observations.

\[\text{Mean} = \bar{x} = \frac{\sum x}{n} = \frac{x_1 + x_2 + ... + x_n}{n}\]

7.3.2 Median

The median is the middle value when data are ranked from lowest to highest.

  • If n is odd: the median is the middle value
  • If n is even: the median is the average of the two middle values

Interpretation: 50% of values lie below the median, 50% lie above.

7.3.3 Mode

The mode is the most frequently occurring value.

Note

The mode is rarely used for numerical data but is useful for categorical data.

7.3.4 Quartiles

Quartiles divide the ordered data into four equal parts:

Quartile Symbol Description
Lower quartile Q1 25% of data below this value
Median Q2 50% of data below this value
Upper quartile Q3 75% of data below this value

7.4 Measures of Dispersion

These describe how spread out the data are around the centre.

7.4.1 Range

The range is simply:

\[\text{Range} = \text{Maximum value} - \text{Minimum value}\]

While easy to calculate, the range is sensitive to outliers.

7.4.2 Standard Deviation (SD)

The standard deviation measures how far observations typically are from the mean.

\[SD = \sqrt{\frac{\sum(x - \bar{x})^2}{n-1}}\]

Steps to calculate SD:

  1. Calculate the mean
  2. Subtract the mean from each value
  3. Square these differences
  4. Sum the squared differences
  5. Divide by (n - 1)
  6. Take the square root

7.4.3 Interquartile Range (IQR)

The interquartile range is the range of the middle 50% of data:

\[IQR = Q3 - Q1\]

The IQR is less affected by outliers than the range or standard deviation.

7.6 Which Measures to Report?

Golden Rule

Always report measures that go together:

  • Mean with standard deviation
  • Median with interquartile range

Never mix: mean with IQR, or median with SD

7.6.1 Choosing Between Mean and Median

Distribution Use Reason
Symmetric (Normal) Mean and SD Mean is appropriate
Skewed Median and IQR Median is more representative
Outliers present Median and IQR Median is less affected
Categorical (mode) Mode Only sensible measure
Show R code 
set.seed(42)
skewed <- tibble(x = rexp(500, 0.5))

mean_val <- mean(skewed$x)
median_val <- median(skewed$x)

ggplot(skewed, aes(x = x)) +
  geom_histogram(bins = 30, fill = "#3498db", colour = "white", alpha = 0.7) +
  geom_vline(xintercept = mean_val, colour = "#e74c3c", linewidth = 1.2, linetype = "dashed") +
  geom_vline(xintercept = median_val, colour = "#27ae60", linewidth = 1.2) +
  annotate("text", x = mean_val + 0.5, y = 60, label = paste("Mean =", round(mean_val, 1)), 
           colour = "#e74c3c", hjust = 0, size = 4) +
  annotate("text", x = median_val + 0.5, y = 70, label = paste("Median =", round(median_val, 1)), 
           colour = "#27ae60", hjust = 0, size = 4) +
  labs(x = "Value", y = "Frequency", 
       title = "Positively skewed distribution") +
  theme_minimal()

In skewed distributions, the mean is “pulled” towards the tail, while the median stays closer to where most data points are.

7.7 Sample Statistics vs Population Parameters

Sample Statistic Population Parameter Symbol
Sample mean (x̄) Population mean μ (mu)
Sample SD Population SD σ (sigma)
Sample median Population median –

We use sample statistics to estimate the unknown population parameters.

7.8 Clinical Application

Reading Research Papers

When you see summary statistics in a paper:

  1. Check whether mean/SD or median/IQR is reported
  2. Consider why the authors chose that measure
  3. If mean/SD is used for skewed data, the results may be misleading
  4. Look at the range to understand the full spread of values

7.9 Summary

  • Mean: sum of values divided by n; best for symmetric data
  • Median: middle value; best for skewed data or when outliers are present
  • Standard deviation: average distance from the mean
  • Interquartile range: spread of the middle 50% of data
  • Always report mean with SD, or median with IQR
  • Choose based on the shape of the distribution

## Worked Example

Let's calculate these measures for SUVmax values from 10 lung cancer patients:

```{r}
#| label: suv-example

# Original data
suv_data <- c(1.8, 8.9, 2.7, 9.4, 5.4, 16.0, 5.8, 17.9, 13.1, 6.6)

# Create a display table
tibble(
  Patient = 1:10,
  SUVmax = suv_data
) %>%
  kable(align = "c")
```

### Step 1: Calculate the Mean

```{r}
#| label: calculate-mean

n <- length(suv_data)
sum_values <- sum(suv_data)
mean_suv <- mean(suv_data)

cat("Sum of all values:", sum_values, "\n")
cat("Number of observations (n):", n, "\n")
cat("Mean = ", sum_values, "/", n, "=", round(mean_suv, 2))
```

### Step 2: Calculate the Standard Deviation

```{r}
#| label: calculate-sd

# Deviations from mean
deviations <- suv_data - mean_suv

# Squared deviations
squared_dev <- deviations^2

# Show calculations
calc_table <- tibble(
  `Original (x)` = suv_data,
  `x - mean` = round(deviations, 1),
  `(x - mean)²` = round(squared_dev, 2)
)

kable(calc_table, align = "c")

# Final calculation
sum_squared <- sum(squared_dev)
variance <- sum_squared / (n - 1)
sd_suv <- sqrt(variance)

cat("\nSum of squared deviations:", round(sum_squared, 2))
cat("\nVariance =", round(sum_squared, 2), "/", n-1, "=", round(variance, 2))
cat("\nStandard deviation = √", round(variance, 2), "=", round(sd_suv, 2))
```

### Step 3: Calculate the Median and Quartiles

```{r}
#| label: calculate-median

# Sort the data
sorted_data <- sort(suv_data)

cat("Ordered data:", paste(sorted_data, collapse = ", "), "\n\n")

# Median (average of 5th and 6th values)
median_suv <- median(suv_data)
q1 <- quantile(suv_data, 0.25)
q3 <- quantile(suv_data, 0.75)

cat("Median (Q2) =", median_suv, "\n")
cat("Lower quartile (Q1) =", q1, "\n")
cat("Upper quartile (Q3) =", q3, "\n")
cat("Interquartile range (IQR) =", q1, "to", q3, "\n")
cat("Range =", max(suv_data) - min(suv_data))
```

### Summary of Results

```{r}
#| label: tbl-summary
#| tbl-cap: "Summary statistics for SUVmax values"

summary_table <- tibble(
  Measure = c("Mean (SD)", "Median (IQR)", "Range"),
  Value = c(
    paste0(round(mean_suv, 2), " (", round(sd_suv, 2), ")"),
    paste0(round(median_suv, 2), " (", round(q1, 1), " - ", round(q3, 1), ")"),
```

## Which Measures to Report?

::: {.callout-important}
## Golden Rule
**Always report measures that go together:**

- Mean **with** standard deviation
- Median **with** interquartile range

**Never mix:** mean with IQR, or median with SD
:::

### Choosing Between Mean and Median

| Distribution | Use | Reason |
|--------------|-----|--------|
| Symmetric (Normal) | Mean and SD | Mean is appropriate |
| Skewed | Median and IQR | Median is more representative |
| Outliers present | Median and IQR | Median is less affected |
| Categorical (mode) | Mode | Only sensible measure |

```{r}

#| fig-width: 8
#| fig-height: 4

  geom_vline(xintercept = mean_val, colour = "#e74c3c", linewidth = 1.2, linetype = "dashed") +
  geom_vline(xintercept = median_val, colour = "#27ae60", linewidth = 1.2) +
  annotate("text", x = mean_val + 0.5, y = 60, label = paste("Mean =", round(mean_val, 1)), 
           colour = "#e74c3c", hjust = 0, size = 4) +
  annotate("text", x = median_val + 0.5, y = 70, label = paste("Median =", round(median_val, 1)), 
           colour = "#27ae60", hjust = 0, size = 4) +
  labs(x = "Value", y = "Frequency", 
       title = "Positively skewed distribution") +
  theme_minimal()
```

In skewed distributions, the mean is "pulled" towards the tail, while the median stays closer to where most data points are.

- **Mean**: sum of values divided by n; best for symmetric data
- **Median**: middle value; best for skewed data or when outliers are present

- **Interquartile range**: spread of the middle 50% of data
- Always report mean **with** SD, or median **with** IQR
- Choose based on the shape of the distribution
