6 Presenting Data

6.1 Why Visualise Data?

Before any statistical analysis, we should look at our data.

Visualisation helps us:

  • Identify the main features of the data
  • Detect outliers (extreme values)
  • Spot data entry errors
  • Choose appropriate statistical methods
What are Outliers?

Outliers are extreme observations that are inconsistent with the rest of the data. They can distort statistical analyses and may indicate measurement errors or genuinely unusual cases.

6.2 Frequency Distribution Tables

A frequency count is simply how many times each value occurs.

6.2.1 Simple Frequency Table

Show R code 
brain_mets <- tibble(
  `Number of Brain Metastases` = c("1", "2", "3 or more", "Total"),
  `Number of Patients (n)` = c(19, 4, 9, 32),
  `Percent of Patients (%)` = c(59.4, 12.5, 28.1, 100)
)

kable(brain_mets, align = "c")

The most frequent category is called the mode. Here, the mode is 1 metastasis (occurring in 19 patients).

6.2.2 Relative Frequencies

Percentages can also be expressed as proportions:

  • 59.4% = 0.594
  • 12.5% = 0.125
  • 28.1% = 0.281

6.2.3 Grouped Frequency Tables

When data have many possible values, we group them into intervals (e.g., age groups: 20-29, 30-39, 40-49, etc.).

6.3 Graphs for Numerical Data

6.3.1 Histograms

A histogram shows the distribution of a continuous variable using bars.

Show R code 
# Simulate height data (approximately normal)
set.seed(42)
heights <- tibble(height = rnorm(5682, mean = 1.61, sd = 0.07))

ggplot(heights, aes(x = height)) +
  geom_histogram(binwidth = 0.05, fill = "#3498db", colour = "white", alpha = 0.8) +
  labs(x = "Height (m)", y = "Frequency (n)") +
  scale_x_continuous(breaks = seq(1.2, 2.0, 0.2)) +
  theme_minimal() +
  theme(
    axis.title = element_text(size = 12),
    axis.text = element_text(size = 10)
  )

Key features of histograms:

  • X-axis: values of the variable
  • Y-axis: frequency (count) or relative frequency (%)
  • Bar width: size of each group/interval
  • Shape reveals the distribution pattern

6.3.2 Understanding Distribution Shapes

Show R code 
set.seed(123)

# Create three different distributions
neg_skew <- tibble(x = rbeta(1000, 8, 2) * 70 + 10, type = "Negatively Skewed\n(long tail to left)")
symmetric <- tibble(x = rnorm(1000, 45, 10), type = "Symmetric\n(bell-shaped)")
pos_skew <- tibble(x = rbeta(1000, 2, 8) * 70 + 10, type = "Positively Skewed\n(long tail to right)")

all_dist <- bind_rows(neg_skew, symmetric, pos_skew) %>%
  mutate(type = factor(type, levels = c("Negatively Skewed\n(long tail to left)", 
                                         "Symmetric\n(bell-shaped)", 
                                         "Positively Skewed\n(long tail to right)")))

ggplot(all_dist, aes(x = x)) +
  geom_histogram(bins = 20, fill = "#3498db", colour = "white", alpha = 0.8) +
  facet_wrap(~type, scales = "free_y") +
  labs(x = "Variable", y = "Frequency") +
  theme_minimal() +
  theme(
    strip.text = element_text(size = 11, face = "bold"),
    axis.title = element_text(size = 11)
  )
The Normal Distribution

A symmetric, bell-shaped histogram is characteristic of the Normal distribution – a fundamental concept we will explore in detail later.

6.3.3 Dot Plots

Dot plots show every individual data point.

Show R code 
set.seed(42)
psa_data <- tibble(
  timepoint = rep(c("Initial", "Week 18"), each = 41),
  psa = c(
    # Initial PSA - higher values, more spread
    c(runif(25, 6, 15), runif(10, 15, 22), runif(6, 22, 30)),
    # Week 18 PSA - lower values, clustered
    c(runif(35, 0, 4), runif(6, 4, 6))
  )
) %>%
  mutate(timepoint = factor(timepoint, levels = c("Initial", "Week 18")))

ggplot(psa_data, aes(x = timepoint, y = psa)) +
  geom_jitter(width = 0.1, size = 2.5, colour = "#2c3e50", alpha = 0.7) +
  labs(x = "", y = "PSA (ng/mL)") +
  theme_minimal() +
  theme(
    axis.title = element_text(size = 12),
    axis.text = element_text(size = 11)
  )

Advantages of dot plots:

  • Shows all individual values
  • Reveals differences between groups clearly
  • Makes outliers visible

6.3.4 Box Plots

A box plot (or box-and-whisker plot) shows five key statistics:

Show R code 
set.seed(42)
example_data <- tibble(x = "Data", y = c(rnorm(95, 5, 1.5), 9.5, 10))

ggplot(example_data, aes(x = x, y = y)) +
  geom_boxplot(fill = "#3498db", alpha = 0.6, width = 0.4, outlier.shape = 1, outlier.size = 3) +
  # Annotations
  annotate("text", x = 1.35, y = 7.25, label = "Upper Quartile (Q3)\n75% below this line", 
           hjust = 0, size = 3.5) +
  annotate("segment", x = 1.22, xend = 1.32, y = 6.5, yend = 6.5, colour = "#2c3e50") +
  annotate("text", x = 1.35, y = 5, label = "Median (Q2)\n50% below this line", 
           hjust = 0, size = 3.5) +
  annotate("segment", x = 1.22, xend = 1.32, y = 5, yend = 5, colour = "#2c3e50") +
  annotate("text", x = 1.35, y = 3.5, label = "Lower Quartile (Q1)\n25% below this line", 
           hjust = 0, size = 3.5) +
  annotate("segment", x = 1.22, xend = 1.32, y = 3.5, yend = 3.5, colour = "#2c3e50") +
  annotate("text", x = 1.35, y = 8.8, label = "Maximum\n(or upper whisker)", hjust = 0, size = 3.5) +
  annotate("text", x = 1.35, y = 1.8, label = "Minimum\n(or lower whisker)", hjust = 0, size = 3.5) +
  annotate("text", x = 0.6, y = 10, label = "Outliers", hjust = 1, size = 3.5) +
  labs(x = "", y = "Values") +
  theme_minimal() +
  theme(axis.text.x = element_blank()) +
  coord_cartesian(xlim = c(0.4, 2))

The five-number summary:

Statistic Symbol Description
Minimum Smallest value (or lower whisker)
Lower quartile Q1 25% of data below this
Median Q2 50% of data below this
Upper quartile Q3 75% of data below this
Maximum Largest value (or upper whisker)

Clinical example:

Show R code 
set.seed(42)
nlr_data <- tibble(
  sex = rep(c("Women", "Men"), c(60, 42)),
  nlr = c(
    c(rnorm(55, 3.2, 1.2), 6.5, 7.5, 8.5, 9.2, 10.5),  # Women with outliers
    c(rnorm(38, 3.8, 1.0), 7.8, 8.2, 9.5, 11.2)  # Men with outliers
  )
)

ggplot(nlr_data, aes(x = sex, y = nlr)) +
  geom_boxplot(fill = "#3498db", alpha = 0.6, width = 0.5, outlier.shape = 1, outlier.size = 2) +
  labs(x = "", y = "Neutrophil-Lymphocyte Ratio") +
  theme_minimal() +
  theme(axis.title = element_text(size = 12))

6.3.5 Scatter Plots

Scatter plots show the relationship between two numerical variables.

Show R code 
set.seed(42)

# Create correlated blood count data
n <- 100
neutrophils <- rnorm(n, 6, 2)
wbc <- neutrophils + rnorm(n, 3, 1.5)  # Strong positive correlation
platelets <- rnorm(n, 250, 80)  # Weak correlation with WBC

blood_data <- tibble(
  neutrophils = pmax(neutrophils, 1),
  wbc = pmax(wbc, 2),
  platelets = pmax(platelets, 50)
)

# Add one outlier
blood_data$wbc[50] <- 18

p1 <- ggplot(blood_data, aes(x = platelets, y = wbc)) +
  geom_point(colour = "#2c3e50", alpha = 0.6, size = 2) +
  labs(x = "Platelets", y = "White Blood Cells") +
  theme_minimal()

p2 <- ggplot(blood_data, aes(x = neutrophils, y = wbc)) +
  geom_point(colour = "#2c3e50", alpha = 0.6, size = 2) +
  labs(x = "Neutrophils", y = "White Blood Cells") +
  theme_minimal()

p1 + p2

The left plot shows a weak positive association. The right plot shows a stronger positive association (unsurprisingly, as white blood cells include neutrophils).

6.4 Graphs for Categorical Data

6.4.1 Bar Charts

Bar charts display the frequency of categories.

Show R code 
stage_data <- tibble(
  stage = factor(c("Ia", "Ib", "IIa", "IIb", "IIIa", "IIIb"), 
                 levels = c("Ia", "Ib", "IIa", "IIb", "IIIa", "IIIb")),
  n = c(8, 5, 10, 6, 42, 22)
)

ggplot(stage_data, aes(x = stage, y = n)) +
  geom_col(fill = "#3498db", width = 0.7) +
  labs(x = "Clinical Stage", y = "Frequency") +
  theme_minimal() +
  theme(axis.title = element_text(size = 12))

Key difference from histograms:

  • Bar charts: gaps between bars (categorical data)
  • Histograms: no gaps (continuous data)

6.4.2 Pie Charts

Pie charts show proportions of a whole.

Show R code 
lobe_data <- tibble(
  lobe = c("Left Upper Lobe", "Left Lower Lobe", "Right Upper Lobe", 
           "Right Lower Lobe", "Right Middle Lobe"),
  n = c(35, 10, 28, 19, 1),
  pct = round(n/sum(n) * 100, 0)
) %>%
  mutate(label = paste0(lobe, "\n(", pct, "%)"))

# Calculate label positions
lobe_data <- lobe_data %>%
  arrange(desc(lobe)) %>%
  mutate(
    cumsum = cumsum(n),
    midpoint = cumsum - n/2,
    pos = midpoint / sum(n) * 2 * pi
  )

ggplot(lobe_data, aes(x = "", y = n, fill = lobe)) +
  geom_bar(stat = "identity", width = 1, colour = "white") +
  coord_polar("y", start = 0) +
  scale_fill_brewer(palette = "Set2") +
  labs(fill = "Tumour Location") +
  theme_void() +
  theme(legend.position = "right",
        legend.title = element_text(face = "bold"))
When to Use Pie Charts

Pie charts work best with 3-5 categories where you want to show parts of a whole. For more categories, bar charts are usually clearer.

6.5 Tables for Bivariate Categorical Data

6.5.1 Contingency Tables

A contingency table (cross-tabulation) shows the relationship between two categorical variables.

Show R code 
contingency <- tibble(
  `Status at Two Years` = c("Alive", "Died", "Total"),
  `Women (n)` = c("48 (80%)", "12 (20%)", "60 (100%)"),
  `Men (n)` = c("29 (69%)", "13 (31%)", "42 (100%)"),
  `Total (n)` = c("77 (75%)", "25 (25%)", "102 (100%)")
)

kable(contingency, align = "c")

Interpretation:

  • 31% of men had died compared to only 20% of women
  • There is a suggestion of a possible association between sex and outcome
Comparing Groups

When groups have different sizes, always use percentages (relative frequencies) rather than raw counts for comparison.

6.5.2 Grouped Bar Charts

Show R code 
set.seed(42)
ps_data <- tibble(
  sex = rep(c("Women", "Men"), c(60, 42)),
  ps = c(
    sample(0:3, 60, replace = TRUE, prob = c(0.15, 0.40, 0.30, 0.15)),
    sample(0:3, 42, replace = TRUE, prob = c(0.35, 0.30, 0.25, 0.10))
  )
) %>%
  count(sex, ps) %>%
  group_by(sex) %>%
  mutate(pct = n / sum(n) * 100)

# Three types of bar charts
p1 <- ggplot(ps_data, aes(x = factor(ps), y = n, fill = sex)) +
  geom_col(position = "dodge", width = 0.7) +
  scale_fill_manual(values = c("#e74c3c", "#3498db")) +
  labs(x = "Performance Status", y = "Count", fill = "", title = "Clustered") +
  theme_minimal() +
  theme(legend.position = "bottom")

p2 <- ggplot(ps_data, aes(x = factor(ps), y = n, fill = sex)) +
  geom_col(width = 0.7) +
  scale_fill_manual(values = c("#e74c3c", "#3498db")) +
  labs(x = "Performance Status", y = "Count", fill = "", title = "Stacked") +
  theme_minimal() +
  theme(legend.position = "bottom")

p3 <- ggplot(ps_data, aes(x = factor(ps), y = pct, fill = sex)) +
  geom_col(width = 0.7) +
  scale_fill_manual(values = c("#e74c3c", "#3498db")) +
  labs(x = "Performance Status", y = "Percent (%)", fill = "", title = "100% Stacked") +
  theme_minimal() +
  theme(legend.position = "bottom")

p1 + p2 + p3 + plot_layout(guides = "collect") & theme(legend.position = "bottom")

Each format has its uses:

  • Clustered: Compare actual counts across groups
  • Stacked: Show totals and composition
  • 100% Stacked: Compare proportions when group sizes differ

6.6 Summary

  • Always visualise data before analysis
  • Choose the right graph for your data type:
Data Type Recommended Graph
Single numerical variable Histogram, box plot
Two numerical variables Scatter plot
Single categorical variable Bar chart, pie chart
Two categorical variables Contingency table, grouped bar chart
  • Look for patterns, outliers, and unexpected features
  • Use percentages when comparing groups of different sizes

### Box Plots

A **box plot** (or box-and-whisker plot) shows five key statistics:

```{r}

example_data <- tibble(x = "Data", y = c(rnorm(95, 5, 1.5), 9.5, 10))

  annotate("text", x = 1.35, y = 1.8, label = "Minimum\n(or lower whisker)", hjust = 0, size = 3.5) +

  labs(x = "", y = "Values") +
  theme_minimal() +
  theme(axis.text.x = element_blank()) +

```

**The five-number summary:**

| Statistic | Symbol | Description |
|-----------|--------|-------------|
| Minimum | | Smallest value (or lower whisker) |
| Lower quartile | Q1 | 25% of data below this |
| Median | Q2 | 50% of data below this |
| Upper quartile | Q3 | 75% of data below this |
| Maximum | | Largest value (or upper whisker) |

### Scatter Plots

**Scatter plots** show the relationship between two numerical variables.

```{r}
#| label: fig-scatterplot
#| fig-cap: "Relationship between blood cell counts: platelets vs white blood cells (left) and neutrophils vs white blood cells (right)"
#| fig-width: 10
#| fig-height: 4.5

set.seed(42)

# Create correlated blood count data
n <- 100
neutrophils <- rnorm(n, 6, 2)
wbc <- neutrophils + rnorm(n, 3, 1.5)  # Strong positive correlation
platelets <- rnorm(n, 250, 80)  # Weak correlation with WBC

blood_data <- tibble(
  neutrophils = pmax(neutrophils, 1),
  wbc = pmax(wbc, 2),
  platelets = pmax(platelets, 50)
)

# Add one outlier
blood_data$wbc[50] <- 18

p1 <- ggplot(blood_data, aes(x = platelets, y = wbc)) +
  geom_point(colour = "#2c3e50", alpha = 0.6, size = 2) +
  labs(x = "Platelets", y = "White Blood Cells") +
  theme_minimal()

p2 <- ggplot(blood_data, aes(x = neutrophils, y = wbc)) +
  geom_point(colour = "#2c3e50", alpha = 0.6, size = 2) +
  labs(x = "Neutrophils", y = "White Blood Cells") +
  theme_minimal()

p1 + p2
```

The left plot shows a **weak positive association**. The right plot shows a **stronger positive association** (unsurprisingly, as white blood cells include neutrophils).

## Graphs for Categorical Data

### Bar Charts

**Bar charts** display the frequency of categories.

```{r}
#| label: fig-barchart
#| fig-cap: "Clinical stage among 93 patients who received radical radiotherapy for non-small cell lung cancer"
#| fig-width: 7
#| fig-height: 5

stage_data <- tibble(
  stage = factor(c("Ia", "Ib", "IIa", "IIb", "IIIa", "IIIb"), 
                 levels = c("Ia", "Ib", "IIa", "IIb", "IIIa", "IIIb")),
  n = c(8, 5, 10, 6, 42, 22)
)

ggplot(stage_data, aes(x = stage, y = n)) +
  geom_col(fill = "#3498db", width = 0.7) +
  labs(x = "Clinical Stage", y = "Frequency") +
  theme_minimal() +
  theme(axis.title = element_text(size = 12))
```

**Key difference from histograms:**

- Bar charts: **gaps between bars** (categorical data)
- Histograms: **no gaps** (continuous data)

### Pie Charts

**Pie charts** show proportions of a whole.

```{r}
#| label: fig-piechart
#| fig-cap: "Tumour location among 93 patients who received radical radiotherapy for lung cancer"
#| fig-width: 7
#| fig-height: 5

lobe_data <- tibble(
  lobe = c("Left Upper Lobe", "Left Lower Lobe", "Right Upper Lobe", 
           "Right Lower Lobe", "Right Middle Lobe"),
  n = c(35, 10, 28, 19, 1),
  pct = round(n/sum(n) * 100, 0)
) %>%
  mutate(label = paste0(lobe, "\n(", pct, "%)"))

# Calculate label positions
lobe_data <- lobe_data %>%
  arrange(desc(lobe)) %>%
  mutate(
    cumsum = cumsum(n),
    midpoint = cumsum - n/2,
    pos = midpoint / sum(n) * 2 * pi
  )

ggplot(lobe_data, aes(x = "", y = n, fill = lobe)) +
  geom_bar(stat = "identity", width = 1, colour = "white") +
  coord_polar("y", start = 0) +
  scale_fill_brewer(palette = "Set2") +
  labs(fill = "Tumour Location") +
  theme_void() +
  theme(legend.position = "right",
        legend.title = element_text(face = "bold"))
```

::: {.callout-tip}
## When to Use Pie Charts
Pie charts work best with 3-5 categories where you want to show parts of a whole. For more categories, bar charts are usually clearer.
:::

## Tables for Bivariate Categorical Data

### Contingency Tables

A **contingency table** (cross-tabulation) shows the relationship between two categorical variables.

```{r}
#| label: tbl-contingency
#| tbl-cap: "Two-year follow-up status by sex among 102 patients following stereotactic radiotherapy for lung cancer"

contingency <- tibble(
  `Status at Two Years` = c("Alive", "Died", "Total"),
  `Women (n)` = c("48 (80%)", "12 (20%)", "60 (100%)"),
  `Men (n)` = c("29 (69%)", "13 (31%)", "42 (100%)"),
  `Total (n)` = c("77 (75%)", "25 (25%)", "102 (100%)")
)

kable(contingency, align = "c")
```

**Interpretation:**

- 31% of men had died compared to only 20% of women
- There is a suggestion of a possible association between sex and outcome

::: {.callout-important}
## Comparing Groups
When groups have different sizes, always use **percentages** (relative frequencies) rather than raw counts for comparison.
:::

### Grouped Bar Charts

```{r}
#| label: fig-grouped-bars
#| fig-cap: "Distribution of performance status by sex among 102 NSCLC patients"
#| fig-width: 10
#| fig-height: 4

set.seed(42)
ps_data <- tibble(
  sex = rep(c("Women", "Men"), c(60, 42)),
  ps = c(
    sample(0:3, 60, replace = TRUE, prob = c(0.15, 0.40, 0.30, 0.15)),
    sample(0:3, 42, replace = TRUE, prob = c(0.35, 0.30, 0.25, 0.10))
  )
) %>%
  count(sex, ps) %>%
  group_by(sex) %>%
  mutate(pct = n / sum(n) * 100)

# Three types of bar charts
p1 <- ggplot(ps_data, aes(x = factor(ps), y = n, fill = sex)) +
  geom_col(position = "dodge", width = 0.7) +
  scale_fill_manual(values = c("#e74c3c", "#3498db")) +
  labs(x = "Performance Status", y = "Count", fill = "", title = "Clustered") +
  theme_minimal() +
  theme(legend.position = "bottom")

p2 <- ggplot(ps_data, aes(x = factor(ps), y = n, fill = sex)) +
  geom_col(width = 0.7) +
  scale_fill_manual(values = c("#e74c3c", "#3498db")) +
  labs(x = "Performance Status", y = "Count", fill = "", title = "Stacked") +
  theme_minimal() +
  theme(legend.position = "bottom")

p3 <- ggplot(ps_data, aes(x = factor(ps), y = pct, fill = sex)) +
  geom_col(width = 0.7) +
  scale_fill_manual(values = c("#e74c3c", "#3498db")) +
  labs(x = "Performance Status", y = "Percent (%)", fill = "", title = "100% Stacked") +
  theme_minimal() +
  theme(legend.position = "bottom")

p1 + p2 + p3 + plot_layout(guides = "collect") & theme(legend.position = "bottom")
```

Each format has its uses:

- **Clustered**: Compare actual counts across groups
- **Stacked**: Show totals and composition
- **100% Stacked**: Compare proportions when group sizes differ

## Summary

- Always **visualise data before analysis**
- Choose the right graph for your data type:

| Data Type | Recommended Graph |
|-----------|-------------------|
| Single numerical variable | Histogram, box plot |
| Two numerical variables | Scatter plot |
| Single categorical variable | Bar chart, pie chart |
| Two categorical variables | Contingency table, grouped bar chart |

- Look for patterns, outliers, and unexpected features
- Use percentages when comparing groups of different sizes
