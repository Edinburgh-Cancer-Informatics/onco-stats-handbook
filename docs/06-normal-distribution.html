8 The Normal Distribution

8.1 Why the Normal Distribution Matters

The Normal distribution (also called the Gaussian distribution) is fundamental to statistics.

Many statistical methods assume data follow a Normal distribution. Understanding it is essential for interpreting medical research.

8.2 Characteristics of the Normal Distribution

The Normal distribution has a distinctive bell shape:

Show R code 
x <- seq(-4, 4, 0.01)
y <- dnorm(x)

ggplot(tibble(x, y), aes(x, y)) +
  geom_area(fill = "#3498db", alpha = 0.4) +
  geom_line(colour = "#2c3e50", linewidth = 1) +
  geom_vline(xintercept = 0, colour = "#e74c3c", linewidth = 1, linetype = "dashed") +
  annotate("text", x = 0.15, y = 0.35, label = "Mean (μ)", hjust = 0, colour = "#e74c3c", size = 4) +
  labs(x = "Value", y = "Probability density") +
  theme_minimal() +
  theme(axis.text.y = element_blank())

Key features:

  • Symmetric around the mean
  • Bell-shaped curve
  • Mean = Median = Mode (all at the centre)
  • Completely described by just two parameters

8.3 The Two Parameters

The Normal distribution is fully defined by:

Parameter Symbol Meaning
Mean μ (mu) Centre of the distribution
Standard deviation σ (sigma) Spread of the distribution
Show R code 
x <- seq(-6, 10, 0.01)

# Different means
p1 <- ggplot() +
  geom_line(aes(x = x, y = dnorm(x, mean = 0, sd = 1)), colour = "#3498db", linewidth = 1) +
  geom_line(aes(x = x, y = dnorm(x, mean = 3, sd = 1)), colour = "#e74c3c", linewidth = 1) +
  annotate("text", x = 0, y = 0.35, label = "μ = 0", colour = "#3498db", size = 4) +
  annotate("text", x = 3, y = 0.35, label = "μ = 3", colour = "#e74c3c", size = 4) +
  labs(x = "Value", y = "Density", title = "Different means (same SD)") +
  theme_minimal()

# Different SDs
p2 <- ggplot() +
  geom_line(aes(x = x, y = dnorm(x, mean = 2, sd = 0.5)), colour = "#3498db", linewidth = 1) +
  geom_line(aes(x = x, y = dnorm(x, mean = 2, sd = 1)), colour = "#e74c3c", linewidth = 1) +
  geom_line(aes(x = x, y = dnorm(x, mean = 2, sd = 2)), colour = "#27ae60", linewidth = 1) +
  annotate("text", x = 2, y = 0.75, label = "σ = 0.5", colour = "#3498db", size = 3.5) +
  annotate("text", x = 2, y = 0.35, label = "σ = 1", colour = "#e74c3c", size = 3.5) +
  annotate("text", x = 2, y = 0.18, label = "σ = 2", colour = "#27ae60", size = 3.5) +
  labs(x = "Value", y = "Density", title = "Different SDs (same mean)") +
  theme_minimal()

p1 + p2

8.4 The 95% Rule

Key Property

In a Normal distribution, approximately 95% of values lie within 1.96 standard deviations of the mean.

This means 95% of the population falls between:

\[\mu - 1.96\sigma \quad \text{and} \quad \mu + 1.96\sigma\]

Show R code 
x <- seq(-4, 4, 0.01)
y <- dnorm(x)

shade_data <- tibble(x, y) %>%
  filter(x >= -1.96 & x <= 1.96)

ggplot() +
  geom_area(data = tibble(x, y), aes(x, y), fill = "grey80", alpha = 0.5) +
  geom_area(data = shade_data, aes(x, y), fill = "#3498db", alpha = 0.6) +
  geom_line(data = tibble(x, y), aes(x, y), colour = "#2c3e50", linewidth = 1) +
  geom_vline(xintercept = c(-1.96, 1.96), colour = "#e74c3c", linetype = "dashed", linewidth = 0.8) +
  annotate("text", x = 0, y = 0.15, label = "95%", size = 6, colour = "#2c3e50", fontface = "bold") +
  annotate("text", x = -2.5, y = 0.05, label = "2.5%", size = 4, colour = "#7f8c8d") +
  annotate("text", x = 2.5, y = 0.05, label = "2.5%", size = 4, colour = "#7f8c8d") +
  annotate("text", x = -1.96, y = -0.03, label = "μ - 1.96σ", size = 3.5, colour = "#e74c3c") +
  annotate("text", x = 1.96, y = -0.03, label = "μ + 1.96σ", size = 3.5, colour = "#e74c3c") +
  labs(x = "Value", y = "Probability density") +
  theme_minimal() +
  theme(axis.text.y = element_blank())

8.4.1 Reference Range

This property gives us the reference range – the range containing 95% of the population.

Example: Height of women

If women’s height is Normally distributed with μ = 1.61 m and σ = 0.07 m:

Show R code 
mu <- 1.61
sigma <- 0.07

lower <- mu - 1.96 * sigma
upper <- mu + 1.96 * sigma

cat("Reference range: ", round(lower, 2), "m to", round(upper, 2), "m\n")
Reference range:  1.47 m to 1.75 m
Show R code 
cat("95% of women have heights in this range")
95% of women have heights in this range

8.5 Sample Estimates

In practice, we don’t know μ and σ. We estimate them from sample data:

Population Parameter Sample Estimate
μ (population mean) x̄ (sample mean)
σ (population SD) SD (sample SD)

For samples from a Normal distribution, approximately 95% of values fall between:

\[\bar{x} - 1.96 \times SD \quad \text{and} \quad \bar{x} + 1.96 \times SD\]

8.6 The Standard Error

When we calculate a sample mean, how precise is it?

8.6.1 Sampling Variability

If we took another sample, the mean would likely be different. This variation is called sampling variability.

8.6.2 Standard Error (SE)

The standard error measures the precision of the sample mean:

\[SE = \frac{SD}{\sqrt{n}}\]

Key insight: As sample size (n) increases, the standard error decreases – larger samples give more precise estimates.

Show R code 
se_table <- tibble(
  `Sample Size (n)` = c(10, 50, 100, 500, 5000),
  `SD` = rep(0.07, 5)
) %>%
  mutate(
    `√n` = round(sqrt(`Sample Size (n)`), 2),
    `SE = SD/√n` = round(SD / sqrt(`Sample Size (n)`), 4)
  )

kable(se_table, align = "c")

8.7 Confidence Intervals

A confidence interval gives a range of plausible values for the population mean.

8.7.1 95% Confidence Interval for the Mean

\[\bar{x} - 1.96 \times SE \quad \text{to} \quad \bar{x} + 1.96 \times SE\]

Interpretation

A 95% confidence interval means: if we repeated the study many times, 95% of the calculated intervals would contain the true population mean.

Example: Women’s height (n = 5,628)

Show R code 
n <- 5628
mean_height <- 1.61
sd_height <- 0.07
se <- sd_height / sqrt(n)

ci_lower <- mean_height - 1.96 * se
ci_upper <- mean_height + 1.96 * se

cat("Sample mean:", mean_height, "m\n")
Sample mean: 1.61 m
Show R code 
cat("Standard error:", round(se, 4), "m\n")
Standard error: 9e-04 m
Show R code 
cat("95% CI:", round(ci_lower, 3), "to", round(ci_upper, 3), "m")
95% CI: 1.608 to 1.612 m

With a large sample, our estimate is very precise (narrow CI).

8.8 Skewed Distributions

Not all data are Normally distributed. Skewed data are common in medicine.

Show R code 
set.seed(42)
volumes <- rexp(88, 0.15)

p1 <- ggplot(tibble(x = volumes), aes(x = x)) +
  geom_histogram(bins = 15, fill = "#3498db", colour = "white", alpha = 0.7) +
  labs(x = "Lesion Volume (cm³)", y = "Frequency", title = "Original data (skewed)") +
  theme_minimal()

p2 <- ggplot(tibble(x = log(volumes)), aes(x = x)) +
  geom_histogram(bins = 15, fill = "#27ae60", colour = "white", alpha = 0.7) +
  labs(x = "Log(Volume)", y = "Frequency", title = "Log-transformed (more symmetric)") +
  theme_minimal()

p1 + p2

8.8.1 Dealing with Non-Normal Data

Approach When to Use
Log transformation Positively skewed data
Square transformation Negatively skewed data
Non-parametric methods When transformation doesn’t help

8.9 The Central Limit Theorem

A Remarkable Result

The Central Limit Theorem states that the distribution of sample means will be approximately Normal, regardless of the original distribution, as long as the sample size is large enough.

This is why many statistical methods work even when individual values are not Normally distributed – we often work with means, not individual values.

8.10 Summary

  • The Normal distribution is symmetric and bell-shaped
  • It is completely described by mean (μ) and standard deviation (σ)
  • 95% of values lie within 1.96 SDs of the mean
  • The standard error measures precision of the sample mean
  • Confidence intervals give plausible ranges for population parameters
  • Skewed data can often be transformed to approximate Normality
  • The Central Limit Theorem means sample means tend to be Normally distributed

## Confidence Intervals

A **confidence interval** gives a range of plausible values for the population mean.

### 95% Confidence Interval for the Mean

$$\bar{x} - 1.96 \times SE \quad \text{to} \quad \bar{x} + 1.96 \times SE$$

::: {.callout-note}
## Interpretation
A 95% confidence interval means: if we repeated the study many times, 95% of the calculated intervals would contain the true population mean.
:::

**Example: Women's height (n = 5,628)**

```{r}
#| label: ci-example

n <- 5628
mean_height <- 1.61
sd_height <- 0.07

```

With a large sample, our estimate is very precise (narrow CI).

## Summary

- The **Normal distribution** is symmetric and bell-shaped
- It is completely described by **mean (μ)** and **standard deviation (σ)**
- **95% of values** lie within 1.96 SDs of the mean
- The **standard error** measures precision of the sample mean
- **Confidence intervals** give plausible ranges for population parameters
- Skewed data can often be **transformed** to approximate Normality
- The **Central Limit Theorem** means sample means tend to be Normally distributed
